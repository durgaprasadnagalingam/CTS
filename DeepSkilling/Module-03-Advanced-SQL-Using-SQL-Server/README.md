\# Module 03 - Advanced SQL Using SQL Server



\## Overview



This module covers advanced SQL Server concepts used for writing optimized queries, creating reusable database logic, handling transactions, and improving database performance.



\## Topics Covered



\- SQL Server fundamentals

\- Window functions

\- OVER() and PARTITION BY

\- ROW\_NUMBER(), RANK(), DENSE\_RANK()

\- GROUPING SETS, CUBE, ROLLUP

\- Common Table Expressions

\- Recursive CTE

\- MERGE statement

\- PIVOT and UNPIVOT

\- Views and Indexes

\- Stored Procedures

\- User Defined Functions

\- Triggers and Cursors

\- Exception Handling

\- Transactions
